# PR Disclaimers

Every comment body — both line-level and summary — posted to a pull request must end with the following disclaimer:

---

> *This review was generated by an AI agent. Findings may be incomplete or incorrect. Use your own judgment before acting on any suggestion.*

---

## Rules

- Append the disclaimer as the **last element** of every comment body before calling any tool that posts to GitHub.
- Do not paraphrase or shorten the disclaimer.
- The disclaimer applies to: PR review comments, PR summary comments, issue close comments, and any other agent-authored comment posted to a remote system.
- It does **not** apply to local output shown only in the terminal.
